あなたはドアの前まで行くとそばに箱があるのに気が付きました。 それには様々な形と大きさの鍵が山のように積まれていました。 エッシャー卿がパズルや複雑さを愛していたのは明らかです。 開けようとしたドアが閉まっていたとしても特に驚くようなことでもありません。 さあ正しい鍵を探し出して先へ進みましょう。

二つの複数行の文字列が入力値です。 一つ目は鍵穴の図です。二つ目は鍵の図です。 あなたの仕事は鍵が鍵穴に合えば True、合わなければ False を返すことです。 鍵と鍵穴は '#' が表し、'0' は鍵の周囲の空間と鍵穴の周りの構造物を表します。 鍵と鍵穴の図は任意の向きで与えられるので注意して下さい。

入力: 鍵穴と鍵を表す複数行の文字列

出力: 真偽値 (True 又は False )

例:

{% if interpreter.slug == "js-node" %} 
keysAndLocks(`
0##0
0##0
00#0
00##
00##`,
`
00000
000##
#####
##000
00000`) == true
{% else %} 
keys_and_locks('''
0##0
0##0
00#0
00##
00##''',
'''
00000
000##
#####
##000
00000''') == True
{% endif %}

どのように使われるか: 二つの物の対応付けを決定できます。

事前条件:
size of key and lock <= 10x10
