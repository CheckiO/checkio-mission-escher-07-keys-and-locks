Подойдя к двери, вы обратили внимание на ящик, который стоял возле нее - он был доверху заполнен ключами всевозможных форм и размеров. Да, лорд Эшер определенно любил головоломки и сложности. Попытавшись открыть дверь, вы, без особого удивления, обнаружили, что она была закрыта. Ну что же - давайте найдем подходящий ключ и двинемся дальше.

В качестве входных данных вы получаете две многострочных строки - одна является схематическим изображение замочной скважины, вторая - ключа. Ваша задача - вернуть True, если ключ подходит к замку и False в ином случае. Ключ и замочная скважина изображены с помощью символов ‘#’, а с помощью ‘0’ изображено пустое пространство вокруг ключа и материал двери, вокруг замочной скважины. Учитывайте, что ключ и замочная скважина могут быть изображены как вертикально, так и горизонтально.

Входные данные: Схематическое изображение ключа и замочной скважины (многострочные строки).

Выходные данные: True или False.

Пример:

{% if interpreter.slug == "js-node" %} 
keysAndLocks(`
0##0
0##0
00#0
00##
00##`,
`
00000
000##
#####
##000
00000`) == true
{% else %} 
keys_and_locks('''
0##0
0##0
00#0
00##
00##''',
'''
00000
000##
#####
##000
00000''') == True
{% endif %}

Как это используется: Для определения соответствия объектов друг другу.

Предусловия:
размер ключа и замка <= 5x5
